# Anti-fraud-robot
This is a Telegram anti-scam bot implemented using the PHP technology stack.  Its features include: blocking messages from scammers listed in the database, submitting evidence of scammers, reviewing scammer data, and providing a historical record lookup function. 

一个用PHP技术栈实现的Telegram防诈机器人，功能包括：拦截在数据库的诈骗犯的消息，提交诈骗犯证据，审核诈骗犯数据，查询历史记录功能。
